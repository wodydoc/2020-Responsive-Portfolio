Takeaways

Initial Challenges

First time pair-programming since college, but this time I was doing it in Spanish, which added a whole other layer of difficulty

Incorporating MapBox, using Geolocation to locate the user and order search results based on proximity, required a lot of attention to detail

Maintaining strict organization and a solid workflow with team, specifically regarding pushing and pulling from GitHub

My Solutions & Learnings

Two heads are better than one - if the driver encounters a hitch with the code, there will be two of them who'll solve the problem

While pair programming, let go of the need for complete code ownership over every implemented feature, in favor of a team-focused ownership concept

The pair should be equally engaged and be participative for the duration of the task at hand, and design always comes last
